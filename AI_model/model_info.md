# YOLOv8 Disease Detection Model

## Purpose

This model was trained to identify diseases in Bell Pepper leaves for precision agriculture applications.

## Model

YOLOv8

## Framework

Ultralytics YOLOv8

## Input Size

256 x 256

## Confidence Threshold

0.7

## Output

The model predicts:

- Diseased Leaf
- Healthy Leaf

## Deployment

The trained model is deployed on:

- Raspberry Pi
- Flask Monitoring Application
- Precision Spraying Drone System

## Application

The prediction results are used to:

1. Detect affected crops
2. Generate visual bounding boxes
3. Trigger spray decisions
4. Assist precision farming operations
